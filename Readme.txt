操作方法

スタートキー		：Enterキー
攻撃			：スペースキー（残弾数がある状態で）
リロード		：スペースキー（残弾数がない状態で）
全体攻撃（スキル）	：Zキー
ゲーム終了		：Escキー

===========================================================
独自に実装したポイント

・一方方向からではなく全方向から敵が出現してくるように
・弾の威力を一定に変更
・弾に弾数制限を設け、弾がなくなったらリロードが必要に
・狙う方向を選べないようにし、回転する砲台の正面に発射されるように変更
・全体攻撃スキルの追加

担当箇所

合わせて、GithubのプルリクエストとスプレッドシートのWBSシートを参考にしてください。

===========================================================
バグリスト修正箇所

・お社への熊野ダメージが毎フレーム発生している
	対応１にて
・お社をスルーする熊がいる
・ゲームが終了できない
	体力を用意して対応
・オブジェクトが残っている
	画面外に出たら消去する
・リザルトがない
	リザルト画面を追加
・スタートボタンとゲーム内の操作に一貫性がない
	キーボードのみでゲームプレイができるように対応
